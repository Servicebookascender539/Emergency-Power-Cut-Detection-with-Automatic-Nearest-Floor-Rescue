# Emergency Power Cut Detection with Automatic Nearest-Floor Rescue

## Description
A smart elevator safety system that detects emergency power failures and automatically moves the lift to the nearest floor for safe evacuation. This system enhances passenger safety using real-time monitoring and automated rescue mechanisms.

## Features
-Detects sudden power failures
- Automatically moves elevator to nearest floor
- Ensures passenger safety during emergencies
- Reduces need for manual intervention

## Technologies Used
- Embedded Systems / Microcontroller
- Sensors
- Control Logic

## Future Improvements
- IoT integration for remote monitoring
- Mobile alerts for emergency situations
- Backup power optimization
